EJEMPLO CSS INTERNO

En este ejemplo, los estilos CSS se definen dentro de la etiqueta style en la sección del documento HTML. Puedes observar cómo se aplican estilos al body, h1 y p, cambiando la fuente,el color de fondo, el color del texto,el tamaño de fuente y la altura de línea. Esto es lo que se conoce como estilos internos en CSS.
